# COVID-19 Power BI Dashboard

## Project Overview
This project analyzes global COVID-19 data using Power BI.

## Dataset Fields
- Country/Region
- Confirmed Cases
- Deaths
- Recovered
- Active Cases
- New Cases
- New Deaths

## Dashboard Features
- Total Cases KPI
- Total Deaths KPI
- Recovery Rate
- Global Map Visualization
- Country-wise Analysis

## Tools Used
- Power BI
- Excel / CSV
- Data Visualization
